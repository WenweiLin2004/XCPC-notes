# XCPC Notes

> 退役 ACMer，三年来的几乎所有笔记，仅以此网站作纪念。

这是一个用 [VitePress](https://vitepress.dev/) 构建的算法竞赛笔记站，收录了三年竞赛生涯里积累的算法模板、知识点和比赛题解。

在线访问：<https://WenweiLin2004.github.io/XCPC-notes/>

## 内容结构

- **算法模板** `docs/templates/`：反复打磨的板子，按专题归档。
- **知识点** `docs/knowledge/`：按数据结构、图论、计算几何、数学、动态规划、字符串、杂项等中文专题分类。
- **竞赛题解** `docs/contests/`：按 ICPC、CCPC、Codeforces、AtCoder、牛客、洛谷等赛事与平台分类。
